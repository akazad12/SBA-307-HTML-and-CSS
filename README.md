# Description:
- A Simulated Financial Services website that includes a sign up page and a login page
- Uses a Nav bar to help user navigate through different pages
- Includes an about page, which includes a table of contacts for the company
- Includes a support page to a gif

## Technologies 
- Uses CSS Flexbox for most of the html
- Bootstrap used for dropdown menu
